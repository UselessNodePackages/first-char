# first-char

[![Build Status](https://travis-ci.org/Inndy/first-char.svg?branch=master)](https://travis-ci.org/Inndy/first-char)

Get first-char from a string

## Installation

`npm install --save first-char`

## Usage

```javascript
require('first-char')

console.log("Foo bar".firstChar)
```
